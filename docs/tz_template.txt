ТЗ

Разработка программного обеспечения опроса приборов учета тепла и газа.

Требования к програмному обеспечению:
- целевая платформа SMH4;
- операционная система linux;
- режим работы 24х7;
- параметризация приложения конфигурационным файлом;
- поддержка протокола modbus для взаимодествия с верхним уровнем;
- поддержка протокола сп-сеть для чтения параметров приборов учета серии х6х;
- поддержка протокола м4 для чтения параметров приборов учета серии х4х;
- поддержка приема/передачи пакетов ip сети;
- поддержка tcp клиента, интерфейс взаимодествия с приборами учета;
- поддержка tcp сервера, интерфейс взаимодествия с верхним уровнем (ядро SMLogic);

Список задач:
- согласование интерфейса взаимодествия с верхним уровнем (ядро SMLogic, карта памяти регистров modbus);
- разработка структуры программного обеспечения;
- реализация функций чтения конфигурационного файла;
- реализация функций сервера modbus-tcp;
- разработка модульных тестов;
- интеграция приложения с верхним уровнем;
- реализация функций tcp клиента;
- реализация функций взаимодествия по протоколу сп-сеть;
- реализация функций взаимодествия по протоколу m4;
- разработка модульных тестов;
- интеграция приложения с приборами учета;
- разработка документации
- поддержка и устранение ошибок
